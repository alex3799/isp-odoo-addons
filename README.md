# MikroTik Odoo Addons

Open-source Odoo 18 modules for ISP operations.

## Modules

- **mikrotik_integration** — MikroTik RouterOS REST API integration for PPPoE subscriber management, IP pool tracking, and service plan enforcement
- **smartolt_integration** — SmartOLT API integration for ONU management, OLT synchronization, and fiber subscriber provisioning
